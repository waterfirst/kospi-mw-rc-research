# MODEL v6 설계 — 회로 모델 × 다요인 온도계 × 거대자금 전류원

**작성:** 2026-06-30 · 베이스: RLC+다이오드(v5) 유지, 파라미터를 다요인으로 확장

## 핵심 아이디어
v5 다이오드 회로의 각 소자를 **단일 US신호가 아니라 다요인 온도계로 구동**하고,
**거대자금(외국인·기관·패시브/블랙록) 흐름을 전류원 I_forced로 정량화**한다.

## 1. 온도계 요인 → 회로 파라미터 매핑
| 회로 소자 | 물리 의미 | 구동 요인(온도계) |
|---|---|---|
| **V_target** (전원) | 균형가 | US지수(SOX·나스닥·S&P) + 달러인덱스 |
| **R / σ_total** (저항·전도) | 회복 속도 | **USD/KRW·한미금리차** — 고환율·금리차↑ → 외인이탈 → 저항↑(회복 느림) |
| **I_forced** (전류원/다이오드) | 강제 매도 충격 | **거대자금 순플로우** (아래 ②) |
| **C** (커패시터) | 개인 흡수력 | 개인 순매수 누적 |

→ v5의 "당일 외/기 부호" 다이오드를, **FX·금리로 modulate되는 거대자금 전류원**으로 일반화.

## 2. 거대자금 흐름 공식 (블랙록/패시브 전술) — 프록시 모델
실제 블랙록 내부공식은 비공개 → **관측 가능한 패시브/외인 프록시로 추정**:

```
I_forced ≈ α·(외국인 KOSPI 순매수)          # 네이버 일별 (실현 플로우)
         + β·(EWY 순유출/거래량)             # iShares Korea ETF = 직접 패시브 프록시
         + γ·(MSCI/FTSE 리밸런싱 캘린더 더미)  # 분기 리밸런싱일 메가플로우(예정)
         + δ·(FX캐리 신호: USD/KRW·금리차)     # 고환율·금리차 → 외인이탈 압력
         + ε·(레버리지 ETF 디레버리징 추정)     # 6/5 $60B, 취약시 $45B (L_program 관성)
```

- **FX캐리(δ)가 7월 핵심:** USD/KRW 1,554 + 금리차 1.5%p → 외인 구조적 매도(월말 아님).
- **리밸런싱(γ):** MSCI 분기(2/5/8/11월말), 반기(5/11) → 대규모 패시브 매매 예정일 사전 포착.
- **EWY(β):** 미국상장 한국 ETF 자금유출입 = 외국인 패시브 의향 선행지표.

## 3. v6 예측식 (시가/종가)
```
시가 = V_prev + open_beta · (V_target − V_prev) / R_eff
종가 = 시가 + 장중[ σ_total·(V_target−V) − I_forced(장중 외/기/EWY) ] dt
       (다이오드 ON = 외+기 동반매도 → 충격 가산)
R_eff = R0 · (1 + FX·금리 페널티)    # 고환율일수록 회복 저항↑
```
