장중 외인·기관 수급 모니터링, 프로그램 증폭기, Flow Nowcast까지 포함한 실전 개편안
Open Engine Flow Nowcast Close Engine Institution Absorption Avalanche Sell
핵심 판단 1
시가는 개선 가능
EWY 단독이 아니라 SOX + KRW + 손상도 + 잔차보정으로 가야 한다.
핵심 판단 2
종가는 장중 수급형
진짜 edge는 장중 외인·기관·프로그램에 있다. 일봉만으로는 불완전하다.
핵심 판단 3
수급 nowcast 가능
완전 예측은 불가하지만, 확률적 사전 추정은 충분히 가능하다.

1. 최종 구조: 3엔진 + 1감시층

Engine A. Open

  • EWY
  • SOX
  • Nasdaq / S&P
  • USDKRW
  • 전일 손상도
→

Engine B. Flow Nowcast

  • 오늘 외인 방향
  • 기관 방어 가능성
  • 프로그램 증폭 위험
→

Engine C. Close

  • 09:10~14:00 체크포인트
  • 장중 가격 구조
  • 실측 수급 반영

2. 시가 엔진 수정안

  • EWY 단독 앵커 신앙 폐기
  • SOX / 환율 / 전일 overshoot 보정
  • 극단일 fat-tail 압축
  • 평범한 날엔 베이스라인을 못 이기는 경우가 있으므로, 큰 날 오차 절감이 우선
open_ret = f(EWY, SOX, SPX, IXIC) - fx_drag - domestic_damage + residual_correction

3. 장중 종가 엔진 수정안

  • 누적 수급보다 수급 가속도 반영
  • 기관 흡수비율 추가
  • 저가 회복률 추가
  • breadth와 삼성/하이닉스 상대강도 분리
  • 프로그램 매도는 증폭기(amplifier)로 취급

4. 장중 수급은 어떻게 모니터링할까

09:00 09:10 09:30 10:30 12:00 12:30 14:00 15:00 직전

기존 09:00 / 10:30 / 12:00 / 12:30 만으로는 외인 급전환, 기관 방어 진입, 프로그램 급가속을 놓칠 수 있다.

구분봐야 할 것의미
외국인누적 순매수, 최근 10~20분 증가율, 전환 시점하방 주도인지, 매도 둔화/매수 전환인지 확인
기관누적 순매수, 저점 부근 방어 유입, 흡수비율외인 매도를 받아내는지 확인
프로그램총합, 인덱스/비인덱스, 급가속하락/상승 증폭기 여부 판단
가격 구조시가 대비 위치, 저가 회복률, breadth, 삼성/하이닉스진짜 회복인지 착시 반등인지 구분

5. 장 시작 전 수급을 미리 예측할 수 있나?

완전 예측은 불가. 하지만 확률적 nowcast는 가능하다.

외인 nowcast 힌트

  • EWY 강세 / SOX 강세
  • 원화 강세(USDKRW 하락)
  • 달러 약세
  • 삼성전자·하이닉스 동조 강세
foreign_score = a1*EWY + a2*SOX - a3*USDKRW + a4*Samsung_RS + a5*Hynix_RS

6. 기관 / 프로그램 nowcast 힌트

기관

  • 전일 급락 과도
  • 전일 기관 소진 여부
  • 반도체 방어 필요성
  • 국내 안정화 신호
inst_score = b1*prior_drawdown - b2*inst_exhaustion + b3*defense_need

프로그램

  • 선물-현물 베이시스
  • 만기일 / ETF 리밸런싱
  • 대형주 급쏠림

7. 최종 플래그 운영안

구분플래그방향

8. 실전 운영 시나리오

  • 개장 전: Open Engine + 외인/기관/프로그램 nowcast 점수 계산
  • 09:10: 첫 실측 수급과 nowcast 부호 비교, mismatch면 신뢰도 하향
  • 10:30: 외인 지속/전환, 기관 흡수 여부 재판정
  • 12:30: 종가 엔진 본 예측, avalanche / absorption / drift 판정
  • 14:00 이후: 프로그램 급가속 여부로 종가 미세조정

9. 한 줄 판정

  • 장중 외인·기관 수급 모니터링은 이미 가능하다.
  • 수급을 미리 맞히는 건 불가능해도, nowcast는 가능하다.
  • 최종 모델은 시가 엔진 + 수급 nowcast + 장중 종가 엔진으로 분리해야 한다.
