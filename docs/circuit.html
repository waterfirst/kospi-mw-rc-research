1. 등가 회로도 (Equivalent Circuit)

KOSPI 지수 V(t)를 RLC 회로의 노드 전압으로 모델링한다. 미국시장 균형가 V_target이 전원, 복원·관성·탄성이 R·L·C, 그리고 외국인·기관 동반매도가 점화하는 스위칭 다이오드 D_sell이 공황 충격 전류원을 도통시킨다.

전원 V_target (미국신호) R_restore 복원저항 L_program 관성 C_individual 커패시터 D_sell 다이오드 + 공황전류

2. 소자별 물리적 의미

V_target
전원 (균형 전압) — 전일 미국시장(S&P500·필라델피아반도체·EWY)을 KOSPI 균형가로 환산. 시장이 수렴하려는 목표 전위.
R_restore
복원 저항 — 기관·개인·ETF/ADR·프로그램·라운드넘버의 병렬 저저항 매수 경로. 작을수록 회복이 빠름 (τ = RC).
L_program
관성 인덕터 — 프로그램매매·레버리지 청산의 운동량. 2차 미분항을 만들어 오버슈트·진동을 가능케 함 (RC엔 없음).
C_individual
개인 커패시터 — 개인 투자자의 축적된 매수 여력. 충전·방전으로 시정수 τ를 결정.
D_sell
스위칭 다이오드 — F<0 AND I<0 일 때만 도통. 외국인·기관 동반매도에서만 공황 전류원 I_shock을 흐르게 함. 단방향성(비대칭).

3. 지배방정식 (Governing ODE)

L·d²V/dt² + R·dV/dt + (V − V_target)/C = −D_sell(t)·K·|F+I| / C_flow
D_sell(t) = 1 iff F(t) < 0 AND I(t) < 0 (외국인 AND 기관 동반매도) = 0 otherwise (다이오드 차단 → 순수 RLC 복원)

키르히호프 해석

다이오드 차단(OFF) 시 우변=0 → 감쇠비 ζ<1이면 저감쇠 RLC로 V_target을 향해 진동 감쇠하며 수렴. 다이오드 도통(ON) 시 우변에 공황 전류원이 주입 → 하락 방향 충격. 순수 RC(1차)는 단조수렴만 가능해 6/26 같은 +5.4% → −5.8% 반전을 재현 불가. L(관성)과 D_sell(비대칭)이 핵심.
