# 2026년 6월 KOSPI 9,000선 붕괴의 회복 동역학
## 구동 RC(맥스웰–바그너) 모델과 병렬-전도 투자주체 분해

**저자:** 첨물 · 분석 Claude · **일자:** 2026-06-25 · 워킹페이퍼/연구기록
*정보·연구 목적. 투자자문 아님.*

---

### 초록
2026년 6월 23일 KOSPI 서킷브레이커 급락(−9.99%) 이후 회복을 맥스웰–바그너 시장 임피던스 모델의 **구동 RC 완화**로 모델링한다. 지수는 직전고점 9,114.55 → 트로프 8,203.84로 급락 후, 2영업일 만에 낙폭의 약 80%를 회복했다(D+1 8,471.02, D+2 8,930.30). 기존 모델이 회복의 게이트로 삼은 **'외국인 순매수 전환'은 데이터로 반증**되며, 이를 **병렬-전도 투자주체 분해**로 대체한다 — 저저항 방전 경로를 연 것은 외국인이 아니라 **기관**이었다. 시총/지수로 커패시턴스를, 실현 가격충격으로 저항을 캘리브레이션하면, 이번 초고속 회복은 **활성층 저항 R_fast의 급락** 때문이지 커패시턴스 C의 증가가 아니다. 실현 시상수는 ~20일 구조 스케일에서 ~1.8일 활성 스케일로 압축됐다.

### 2. 외국인 게이트 모델의 실패
v1은 '외국인 순매수 전환'을 회복의 준-필요조건(P2/H1)으로 봤으나 데이터로 반증된다. 2년 12개 라운드넘버 낙폭 이벤트:
- 외국인 순매수 전환 = 트로프 후 **중앙값 4영업일**(최대 18일), 회복기간과 상관 **−0.17**(무의미·역효과).
- 기관 순매수 전환 = **중앙값 1일**.
- 반례(2026-05-15): 6일 만에 회복, 외국인은 18일째까지 미전환.

이번 에피소드도 외국인이 3일 내내 순매도였으나 V 회복. **외국인을 기다린 것은 직렬회로 오판이다.**

### 3. 수정 모델(v3): 구동 RC 완화
지배방정식:
```
C_eff(t)·dV/dt = σ_total(t)·[V_target − V(t)] − I_forced_sell(t)
```
해: `V(t) = V_target − (V_target − V₀)·exp(−t/τ)`, `τ = C_eff/σ_total`.

**전도율은 병렬 합산(핵심 교정):**
```
σ_total = σ_F + σ_I + σ_P + σ_ETF + σ_round + σ_program
```
외국인 σ_F가 꺼져도 기관(σ_I)·ETF·라운드넘버 되감기(σ_round)·프로그램 경로가 σ_total을 유지한다. 외국인 순매수는 진입 게이트에서 **H2 신호(9,000 위 지속보유 판단)**로 강등.

**유효 커패시턴스 방전:**
```
C_eff = C_cap − C_retail_discharge − C_forced_unwind
```
6/23 개인 +8.59조 흡수=충전, 이후 개인 매도전환=방전 → ε_eff↓. τ=ε_eff/σ_total이 분자·분모 양쪽에서 감소.

### 4. RC 캘리브레이션 (수치)
- **구조 커패시턴스:** C_struct = 시총/지수 = 7,310.4조/8,930.30 = **0.819조/pt** (≈$0.593B/pt).
- **활성(flow) 커패시턴스:** 6/24 기준 C_flow = (기관+개인 순매수)/지수상승 = 4.5407조/267.18pt = **0.017조/pt** (≈170억/pt). 시총의 ~2.1%만 거래로 가격을 움직임.
- **2-브랜치 RC 사다리**(각 층 고유 R·C·τ):
  - 빠른/활성: C_fast=0.017, τ_fast≈1.25일 ⇒ R_fast≈73.6 (얇음→고R, 微C)
  - 느린/구조: C_struct=0.819, τ_slow≈20.5일 ⇒ R_slow≈25.0 (깊음→저R, 巨C)
- **R_fast 하락 동인(데이터):** 위기→평시 λ ×1.8 하락(1.15→0.64 %/조); P4 하락/상승 비대칭 λ_down/λ_up≈1.47(회복이 구조적으로 저R); 기관 병렬경로 ON(+1.91→+3.32조).

### 5. 왜 빨랐나
실현 τ ≈ **1.77일**(전고점 목표, D+1·D+2 지수피팅, R²=0.90). KOSPI 구조 기준선 ~20.5일 대비 11.6배, S&P 대비 ~3배 빠름. 분해하면 **C는 안 커졌다**(KOSPI C는 여전히 S&P의 ~1/15) — **R_fast가 급락**하고 개인 커패시터가 방전했다. 회로적으로 기관이 외국인 개방채널을 우회하는 저저항 병렬 브랜치를 열었다.

τ는 목표점에 따라:
| 목표 | τ |
|---|---|
| 9,000 단기 재탈환 | 0.8–1.5일 |
| 전고점 9,114.55(헤드라인) | **1.77일** |
| 오버슈팅 9,253–9,385 | 2.2–2.6일 |
