> ## 🎯 3줄 요약 (심사위원용)
> 1. **무엇**: KOSPI를 전기회로로 등가모델링해 매일 **시가(07:30)·종가(12:30)를 점예측**하고, "왜 이 값인지"를 **회로 소자 언어로 설명**하는 MCP 플러그인.
> 2. **차별점**: 챗봇의 그럴듯한 서사가 아니라 — **2026년 7월 실전에서 매일 ex-ante 고정 제출·채점으로 검증**된 정량 임계값(k=0.58, 항복 −30k/−20k)이 코드에 박혀 있고, **실패까지 커밋 단위로 공개**한다.
> 3. **핵심 발견**: ①시가는 미국 지수 재구성이 아닌 **EWY 직독**(gap=0.58×EWY, 소급 3/3일 ≤20pt) ②종가 붕괴는 뉴스가 아닌 **수급의 제너 항복**(외인≤−30k & 순수급≤−20k면 기관 매수 무관) ③반등은 **양방향 애벌란치**(기관 폭주 배증 시 고가캡 해제).

## 📋 어떻게 작동하나 (하루 절차)

```
07:30  forecast_open   전일종가 + EWY 오버나잇 → gap=0.58×EWY (+잔차 되돌림) → 시가 고정 제출
09:00 / 10:30 / 12:00  장중 스냅샷 3회 (외인·기관 수급 임계 추적)
12:30  forecast_close  양방향 애벌란치 사다리 → 종가 고정 제출
15:30~ score_prediction 채점 + explain_regime 회로 복기 → 실패는 당일 코드 반영
```

### 시가 로직 (`forecast_open`)
1. `gap = 0.58 × EWY_overnight` (T_EWY 변압기 결합)
2. 전일 오버슈트 보정: `gap += -0.5 × (전일KOSPI등락 − 0.58×전일EWY)` (변압기 이중계산 방지)
3. SOX 교차검증: `|gap − 0.5×SOX| > 2%p`면 두 값 절충
4. 서사 악재(`hyper_bear`)면 상방 차단(≤−1.0%)
5. 밴드 클램프 ±6%, `시가 = 전일종가 × (1+gap/100)`

### 종가 로직 (`forecast_close`) — 우선순위 사다리
1. **하방 항복**: `외인≤−30k AND 순수급≤−20k` → `min(현재,저가) − 0.5×(현재−저가)` *(기관 매수 무관)*
2. **상방 항복**: `기관≥+20k AND 전일 대비 배증` → `현재 + 0.8×(현재−시가)` *(고가캡 해제, 전일값 없으면 오발화 방지 미발화)*
3. **gap 실패**: `현재<시가−80 AND (외인<−10k or 프로그램<−8k)` → `현재 − 0.4×(현재−저가)`
4. **강기관 드리프트**: `현재≥시가−80 AND 기관>+15k` → `현재 + 0.35×(현재−시가)`
5. **고변동 드리프트**: 그 외 → `현재 + 0.25×(현재−시가)` (현재가 앵커 폐지)

### 정보 부족 시 행동 (환각 방지)
- 라이브 데이터(네이버 금융) 수집 실패 시 **임의 추정하지 않고** `{"error":…, "action":"snapshot 직접 입력 필요"}` 반환.
- 필수 입력 누락 시 어떤 키가 필요한지 명시.
- 모든 예측에 `reasons`(근거 리스트)와 `disclaimer` 동봉.

## 🔍 왜 이 문제인가 — 사용자가 실제로 막히는 세 지점

| 개인 투자자의 오해 | 실전에서 깨진 날 | 회로 소자 해법 |
|---|---|---|
| "미국이 올랐으니 한국도 오르겠지" | 6/30 (미국 재구성 실패) | **T_EWY** — EWY 직독 |
| "기관이 사니 버티겠지" | 7/1·7/2 (부호만 보다 붕괴) | **D_av + G_inst** — 크기·임계 |
| "급락 뒤 반등 = 바닥" | 7/2 데드캣 401pt 참사 | **panic override** 최상단 |

손실은 정보 부족이 아니라 **해석 실패와 타이밍**에서 온다. 이 세 오해는 전부 실전에서 점수를 잃으며 검증한 실패이고, 그 실패가 지금의 소자다.

## ✅ 검증 방법

- **실전 채점**: ex-ante 고정 제출 → 실제값 티어 채점 (위 성적표). 정상 케이스(6/30 9/10)와 예외 케이스(7/2 서킷브레이커, 7/3 V자)를 모두 검증에 포함.
- **단위테스트 4종** (네트워크 불필요): T_EWY 계수 · 하방 항복 · 상방 돌파 · 티어 채점 — `sample_outputs/examples.json`에 실행 예시 동봉.
- **순수함수/IO 분리**: `predict_*`, `score`, `explain_regime`은 네트워크 없이 동작 → 재현 가능.

## ⚠️ 한계 (정직한 각주)
