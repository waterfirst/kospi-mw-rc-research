# KOSPI 다이오드 회로 예측 MCP (`kospi-diode`) v7.1

> KOSPI를 **전기회로**로 등가모델링해 매일 **시가(07:30)·종가(12:30)** 를 점예측하고,
> "왜 이 값인지"를 **회로 소자 언어**로 설명하는 MCP 플러그인.
> 삼성디스플레이 구동회로 엔지니어의 시선으로 시장을 회로로 본다 —
> *주가지수=전압, 매수세=전류.*
>
> **정보·연구 목적. 투자자문 아님.**

---

## 1. 무엇을 하나 (한 줄)

미국 밤 시장(EWY·SOX)과 장중 수급(외국인·기관)을 **회로 소자로 자동 환산**해
"오늘 한국장 시가는 몇 %갭, 종가는 어디"를 숫자와 근거로 답한다.
"오를까/내릴까"가 아니라 **"지금 왜 이 레짐인가"** 를 설명하는 것이 목표다.

## 2. 왜 회로 모델인가

챗봇이 그럴듯한 서사를 생성하는 대신, **시장 신호를 정량 소자로 표준화**한다.
개인 투자자가 막히는 지점은 정보 부족이 아니라 **정보 해석·타이밍**이다:

| 사용자가 막히는 곳 | 회로 소자 해법 |
|---|---|
| 미국은 올랐는데 왜 한국은 빠지지? | **T_EWY 변압기** — 미국 지수 재구성 대신 EWY 직독 |
| 외국인이 파는데 기관이 사니 버티겠지? | **D_av 제너 다이오드** — 임계 초과 시 항복(붕괴) |
| 급락 뒤 반등이 바닥인가 데드캣인가? | **양방향 애벌란치** — 현재가 앵커 폐지·드리프트 연장 |

## 3. 회로 소자 사전

| 소자 | 시장 의미 | 판단 기준(상수) |
|---|---|---|
| **T_EWY** (변압기) | 미국시간 한국 가격발견 = 밤새 채점된 답안지 | `gap = 0.58 × EWY 오버나잇` |
| **V_semi** (전압원) | 반도체 1차 전류원은 하이퍼스케일러(Meta/MSFT/GOOG/AMZN) capex | SOX 교차검증(2%p 괴리시 절충) |
| **D_av** (제너 다이오드) | 수급 임계 초과 시 눈사태 도통, **양방향** 항복 | 하방 외인≤-30k & 순수급≤-20k / 상방 기관≥+20k & 배증 |
| **G_inst** (가변 컨덕턴스) | 방어는 부호(사/팔)가 아닌 **크기**(연속) | 강방어 기관>+15k |
| **R_eff(τ)** (가변 저항) | 위기 레짐서 회복 시정수 2.3→10.3일(×4.4) | VKOSPI≥40 위기 |
| **SW_hyper** (서사 스위치) | 하이퍼스케일러 capex 악재 감지 시 상방 차단 | `hyper_bear` 플래그 |

## 4. 어떻게 작동하나 (하루 절차)

```
07:30  forecast_open   전일종가 + EWY 오버나잇 → gap=0.58×EWY (+잔차 되돌림) → 시가 고정 제출
09:00 / 10:30 / 12:00  장중 스냅샷 3회 (외인·기관 수급 임계 추적)
12:30  forecast_close  양방향 애벌란치 사다리 → 종가 고정 제출
15:30~ score_prediction 채점 + explain_regime 회로 복기 → 실패는 당일 코드 반영
```

### 4.1 시가 로직 (`forecast_open`)
1. `gap = 0.58 × EWY_overnight` (T_EWY 변압기 결합)
2. 전일 오버슈트 보정: `gap += -0.5 × (전일KOSPI등락 − 0.58×전일EWY)` (변압기 이중계산 방지)
3. SOX 교차검증: `|gap − 0.5×SOX| > 2%p`면 두 값 절충
4. 서사 악재(`hyper_bear`)면 상방 차단(≤-1.0%)
5. 밴드 클램프 ±6%, `시가 = 전일종가 × (1+gap/100)`

### 4.2 종가 로직 (`forecast_close`) — 우선순위 사다리
1. **하방 항복**: `외인≤-30k AND 순수급≤-20k` → `min(현재,저가) − 0.5×(현재−저가)` *(기관 매수 무관)*
2. **상방 항복**: `기관≥+20k AND 기관≥2×직전` → `현재 + 0.8×(현재−시가)` *(고가캡 해제)*
3. **gap 실패**: `현재<시가−80 AND (외인<-10k or 프로그램<-8k)` → `현재 − 0.4×(현재−저가)`
4. **강기관 드리프트**: `현재≥시가−80 AND 기관>+15k` → `현재 + 0.35×(현재−시가)`
5. **고변동 드리프트**: 그 외 → `현재 + 0.25×(현재−시가)` (현재가 앵커 폐지)

### 4.3 정보 부족 시 행동
- 라이브 데이터(네이버 금융) 수집 실패 시 **임의 추정하지 않고** `{"error":..., "action":"snapshot 직접 입력 필요"}` 반환.
- 필수 입력 누락 시 어떤 키가 필요한지 명시.
- 모든 예측에 `reasons`(근거 리스트)와 `disclaimer` 동봉.

## 5. 툴 목록

| 툴 | 입력 | 출력 |
|---|---|---|
| `forecast_open` | snapshot{prev_close, ewy_overnight, ...} 또는 없음(라이브) | 시가 예측 + 근거 |
| `forecast_close` | snapshot{open,current,high,low,foreign,inst,...} 또는 없음 | 종가 예측 + 레짐 |
| `explain_regime` | snapshot{ewy_overnight,sox_overnight,foreign,inst,vkospi} | 소자별 해석 |
| `score_prediction` | predicted, actual | 오차율 티어(0~5) |
| `daily_workflow` | 없음 | 하루 운영 루틴 |
| `submission_answers` | 없음 | AX 해커톤 5문항 답변 |

## 7. 검증 (실전 채점)

2026년 7월 초 KOSPI 실전에서 **ex-ante(장전) 고정 제출** 후 실제값 채점:

| 날짜 | 실제 시가/종가 | 이 모델 | 점수 |
|---|---|---|---|
| 6/30 | 8,417 / 8,476 | 8,435 · 8,510 | **9/10** (0.21% / 0.40%) |
| 7/2 | 7,933 / 7,648 | Meta 쇼크 −7.89% | 0/10 → panic override 소급 73pt 개선 |
| 7/3 | 7,740 / 8,088 | V자 +5.76% | 상방 애벌란치 발견·코드화 |

순수 함수는 네트워크 없이 도는 **단위테스트 4종**(T_EWY 계수·하방 항복·상방 돌파·티어 채점)으로 검증. `sample_outputs/`에 실행 예시 동봉.
