# 회복·타이밍 모델 v2 (전면 수정) — 외국인 게이트 폐기
2026-06-25 · 사유: "외국인 순매수를 기다리는 것은 모델의 실패다"(검증됨)

## 1. 폐기된 것 (v1의 치명적 결함)
- v1 마스터 스위치 = "외국인 순매수 전환" / P2·H1의 "외국인 스트릭 소멸 ≈ 회복의 필요조건".
- **데이터 반증(2년 13이벤트):**
  - 외국인 순매수 전환 중앙값 **4일**(최대 18일), 회복일과 상관 **−0.17** → 무의미·지연지표.
  - 기관 전환 중앙값 **1일**(거의 항상 D+1 선행).
  - 2026-05-15: **6일 만에 회복, 외국인은 D+18에야 전환** → 외국인 기다리면 회복 전체를 놓침.
  - 이번 6/23 에피소드: 외국인 3일 내내 순매도, 그래도 2일 +8.8% V(기관 +3.32조 주도).
- **결론: 외국인 순매수는 후행·비필요 지표. 게이트에서 영구 제거.**

## 2. v2 마스터 스위치 (다주체·가격우선)
우선순위 순:
1. **실현 τ (가격 자체)** — 매 세션 회복곡선에 dev(t)=Δ₀·exp(−t/τ) 적합. **τ<3일=빠른V(추세추종·즉시참여), τ>10일=완만(역추세·분할대기).** 가격이 1차 진실.
2. **기관 전도 전환(확인지표)** — 기관 순매수 플립(중앙값 D+1). 있으면 빠른V 확정 가속. **게이트 아님, 확인용.**
3. **외국인 매도 '감속'(부호 아님, 크기)** — Δ외국인매도 < 0(매도 축소)면 σ 부분복원 = 순풍. 부호 전환은 안 기다림.
4. **간밤 美 SOX/반도체(선행)** — 익일 KOSPI 수급·갭 방향 1차 결정자(폐루프).
- **금지: 외국인 순매수 부호를 진입 조건으로 쓰지 말 것.**

## 3. P2·H1 재정의
- **P2(재정의):** 전도 복원은 "국내 전도자본(기관) 흡수"로 충분. 외국인 반전은 *충분조건이지 필요조건 아님*. 지수 매도는 여전히 비차익(바스켓) 채널.
- **H1(재정의):** 바닥은 외국인 반전 없이도 **기관 흡수 + 가격 지지 확인**으로 성립.

## 4. 진입 규칙 (v2)
- 트로프 후 **가격이 D+1에 직전저점 위 + 기관 순매수** → 즉시 1차 진입(35%). 외국인 불문.
- τ<3 추세 지속 → 피라미딩. τ가 늘면(>10) 분할·대기.
- 손절: V구조 깨짐(직전 핵심레벨 종가 이탈). 인밸리데이션은 가격으로 정의, 수급으로 미루지 않음.

- **06:00 KST (美 마감 후):** 간밤 SOX/마이크론·S&P → **KRX 당일(개장 09:00, +3h / 종가 +9h) 갭·경로 예측.**
